## [Demo](https://kenlee100.github.io/hexschool-vue-dashboard)

#### 功能
- 登入、登出
- 行程管理
- 文章管理
- 優惠券管理
- 訂單列表

#### 專案技術
- Vue 3.2.45
- Vite 4.0.0
- Pinia 2.0.28
- Vue Axios 3.5.2
- Vue Router 4.1.6
- Vee-validate 4.7.4
- Vue Loading Overlay 6.0.2
- SweetAlert2 11.7.3
- ckeditor5-vue 4.0.1
- ckeditor5-build-classic 36.0.1
- Bootstrap 5.2.3
- SCSS

#### 安裝套件
```sh
npm install
```
#### 運行專案
```sh
npm run dev
```

#### 開啟專案
在瀏覽器網址列輸入以下即可看到畫面
```sh
http://localhost:3001/
```



#### 打包專案
```sh
npm run build
```

#### 部署專案
```sh
sh deploy.sh
```
